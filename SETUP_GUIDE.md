## 📋 ИНСТРУКЦИЯ ПО УСТАНОВКЕ

### Шаг 1️⃣: Проверьте Python

Откройте **Command Prompt (cmd.exe)** и выполните:

```bash
python --version
```

**Должно вывести:** `Python 3.10.x` или выше

❌ Если ошибка `python: command not found`:
- Скачайте Python с https://www.python.org/downloads/
- **ВАЖНО**: при установке отметьте ✅ "Add Python to PATH"
- Перезагрузитесь и повторите проверку

---

### Шаг 2️⃣: Установка зависимостей

Откройте **Command Prompt** и перейдите в папку проекта:

```bash
cd "c:\моя\ИИ-агент маринет"
```

**Вариант A (автоматически):**
```bash
setup.bat
```
Скрипт автоматически установит всё.

**Вариант B (вручную):**
```bash
pip install -r requirements.txt
```

⏳ Установка займет 2-5 минут (зависит от интернета)

---

### Шаг 3️⃣: Подготовка данных

1. Создайте файл **`data/organizations.xlsx`**
   - Используйте Excel или LibreOffice Calc
   - Скопируйте таблицу организаций из архитектуры (первые 50-100 строк)
   - Сохраните как `.xlsx`

2. Требуемые колонки:
   - **Название** - название организации
   - **Сайт** - полный URL (https://...)
   - **Категории** - категории через запятую

Пример:
```
Название                  | Сайт                      | Категории
АКВАИР ИНЖИНИРИНГ        | https://aquairy.ru        | Мониторинг инфраструктуры, Чрезвычайные ситуации
3Д-Технологии.РУ        | https://3d-tech.ru        | Мониторинг инфраструктуры, Наука и образование
```

---

### Шаг 4️⃣: Запуск агента

```bash
python main.py
```

Агент начнет:
1. ✓ Загружать организации из Excel
2. ✓ Сканировать сайты (поиск RSS, /news, и т.д.)
3. ✓ Парсить новости
4. ✓ Классифицировать по темам
5. ✓ Сохранять в `news_output/`

**Ожидаемый вывод:**

## 🔧 Управление конфигурацией

### Изменить темы классификации

Отредактируйте `config.py`:

```python
GLOBAL_TOPICS = {
    "Моя новая тема": ["ключевое_слово1", "ключевое_слово2"],
    # ... другие темы
}
```

### Изменить формат вывода

В `config.py`:

```python
OUTPUT_FORMAT = "markdown"  # или "txt", "json"
```

### Изменить частоту запусков

```python
SCHEDULE_INTERVAL = "daily"  # или "weekly", "manual"
SCHEDULE_TIME = "09:00"      # время запуска (HH:MM)
```

## ⚠️ Решение проблем

### ❌ "ModuleNotFoundError: No module named 'pandas'"

```bash
pip install pandas openpyxl
```

### ❌ "SSL: CERTIFICATE_VERIFY_FAILED"

Это нормально для некоторых сайтов. Код использует `ssl=False` по умолчанию.

### ❌ "No organizations loaded"

- Проверьте, что файл `data/organizations.xlsx` существует
- Проверьте названия колонок (должны быть на русском: "Название", "Сайт", "Категории")
- Смотрите логи в `logs/news_agent.log`

### ❌ "No news sources found"

- Возможно, сайт использует JavaScript → нужен Selenium
- Проверьте вручную: есть ли RSS-лента на сайте?
- Добавьте свои паттерны в `NEWS_PATTERNS` в `config.py`

### ❌ Долгий запуск

Это нормально! Агент:
- Делает HTTP-запросы к каждому сайту (может быть медленно)
- Парсит HTML (требует времени)
- Для 100+ организаций может работать 5-10 минут
