## 🚀 QUICK START (3 Steps)

### Step 1: Install Python
Download from https://www.python.org/downloads/ (3.10+)
**Check:** `python --version`

### Step 2: Install Dependencies
```bash
cd "c:\моя\ИИ-агент маринет"
setup.bat
```
Or manually:
```bash
pip install -r requirements.txt
```

### Step 3: Run the Agent
```bash
python main.py
```
