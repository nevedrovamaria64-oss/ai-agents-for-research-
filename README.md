# News Agent - Архитектура ИИ-агента для мониторинга новостей

## Установка и запуск

### 1. Установка Python

Убедитесь, что установлен **Python 3.10+**:
```bash
python --version
```

### 2. Установка зависимостей

Откройте **Command Prompt (cmd.exe)** или **PowerShell** и выполните:

```bash
cd "c:\моя\ИИ-агент маринет"
pip install -r requirements.txt
```

Это установит все необходимые библиотеки:
- `pandas` - работа с Excel
- `aiohttp` - асинхронные HTTP-запросы
- `beautifulsoup4` - парсинг HTML
- `feedparser` - парсинг RSS
- И другие...

### 3. Подготовка данных

Создайте файл `data/organizations.xlsx` с организациями. Требуемые колонки:
- **Название** - название организации
- **Сайт** - URL сайта (https://...)
- **Категории** - категории (через запятую)

Пример:
```
Название | Сайт | Категории
3Д-Технологии.РУ | https://3d-tech.ru | Мониторинг инфраструктуры, Наука и образование
```

### 4. Запуск агента

```bash
python main.py
```

Агент начнет:
1. Загружать организации из Excel
2. Сканировать их сайты в поиске новостей
3. Парсить новости (HTML + RSS)
4. Классифицировать по категориям
5. Сохранять в `news_output/` по темам

### 5. Результаты

После завершения результаты будут в папке `news_output/`:
```
news_output/
├── Мониторинг инфраструктуры/
│   ├── 2026-05-21_Организация_Заголовок.md
│   ├── 2026-05-21_Организация_Заголовок.md
├── Энергетический сектор/
│   ├── 2026-05-21_Организация_Заголовок.md
├── index.json (полный индекс всех новостей)
```

## Конфигурация

Отредактируйте `config.py` для изменения:
- **GLOBAL_TOPICS** - список тем и ключевые слова
- **NEWS_PATTERNS** - паттерны поиска новостей (/news, /blog, и т.д.)
- **RSS_PATTERNS** - паттерны RSS-лент
- **SCHEDULE_INTERVAL** - частота запусков
- **OUTPUT_FORMAT** - формат вывода (markdown, txt, json)

## Решение проблем

### 1. "ModuleNotFoundError: No module named 'pandas'"
```bash
pip install pandas openpyxl
```

### 2. "SSL: CERTIFICATE_VERIFY_FAILED"
Используется `ssl=False` в запросах (рискованно для production, но сойдет для тестирования)

### 3. Сайт не скачивается
- Проверьте URL в Excel
- Проверьте доступность сайта вручную в браузере
- Смотрите логи в `logs/news_agent.log`

### 4. Новости не найдены
- Возможно, сайт использует JavaScript (нужен Selenium/Playwright)
- Проверьте, есть ли RSS-лента на сайте
- Добавьте свои паттерны в `NEWS_PATTERNS`
